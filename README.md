# 🌐 iDigital - Business Website for WhatsApp API Integration

## 📌 Overview

iDigital is a business website developed to establish an official online presence required for WhatsApp Business API approval via Meta.

This website was used as **business verification proof**, enabling access to WhatsApp API services, which were later integrated into the iDigital mobile application to automate messaging for student communication.

---

## 🎯 Problem Statement

To use WhatsApp API for automated messaging, Meta requires:

* A verified business presence
* A live and accessible website

This project solves that requirement by providing a deployed business website.

---

## 🚀 Key Features

* 🌍 Live business website deployed on IONOS
* ✅ Used as verification proof for WhatsApp Business API approval
* 💬 Enabled WhatsApp API integration in mobile app
* 📱 Automated messaging system for student notifications
* ⚡ Lightweight and fast static website

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS
* **Hosting:** IONOS
* **API Integration:** WhatsApp Business API (Meta)

---

## 🔗 System Workflow

1. Developed and deployed business website using IONOS
2. Submitted website as business proof to Meta
3. Obtained WhatsApp API access
4. Integrated API into iDigital mobile application
5. Enabled automated messaging to registered student mobile numbers

---

## 📱 Real-World Use Case

This system is used in the iDigital mobile application to:

* Send notifications to students
* Automate communication workflows
* Improve engagement without manual effort
